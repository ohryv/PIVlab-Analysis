# PIVlab-Analysis
Files used for SSR 2026 to analyze PIVlab results in Python

HowManyVectorsAreReal-- Working notebook for finding VDP (valid detection probability) for each frame in an analyzed set of PIVlab images

validate-and-plot-validations-PIVlab-- Finalized .py file of HowManyVectorsAreReal

VelocityAttenuation_WorkingNotebook-- Working notebook for evaluating the velocity attenuation ratio of a PIV dataset. Explores several options in defining the local and streamline regions.
